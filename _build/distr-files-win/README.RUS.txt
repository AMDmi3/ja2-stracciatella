JA2-Stracciatella 0.11 скомпилированные файлы


После того как вы в первый раз запустите игру, в вашей директории с документами
("Мои документы") появится папка "JA2".  В ней есть файл "ja2.ini".
Отредактируйте этот файл, прописав в нем правильный путь к файлам данных Jagged
Alliance 2 в строке "data_dir = [...]".  Назовем папку с игровыми данными $DIR,
тогда в ней должно быть следующее:
- $DIR/Data/*.slf           - все файлы .slf с инсталляционного И игрового CD
- $DIR/Data/TILECACHE/*.sti - все файлы из директории TILECACHE инсталлированной
- $DIR/Data/TILECACHE/*.jsd   игры

Общее количество должно быть
-  22 .slf (или 23, зависит от версии локализации)
-  28 .sti
- 109 .jsd
